FAQ

Do you use WordPress or website builders?
No. Pure HTML, CSS and JavaScript only. Your site will be fast, clean and you'll own every line of code — no platform lock-in, no monthly fees.
Can I edit the site myself after you build it?
Yes. I write readable, well-commented code. I'll also give you a short guide on how to make common updates yourself.
What if my site is already built and just needs fixes?
The Patch package is for you. Share your code via GitHub or a ZIP file and describe what's broken.
Do you do React or other frameworks?
Yes — React, Tailwind CSS and other modern frameworks are supported. For smaller sites vanilla HTML/CSS/JS is often faster and cleaner. For larger apps and dashboards React is the right tool. Tell me what you need and I'll recommend the right stack.
